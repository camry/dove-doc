# TCP 服务器

TCP 服务器。

## 注册服务

```go
import (
    ggtcp "github.com/camry/g/gnet/gtcp"
)

tcp := gtcp.NewServer(gtcp.Handler(func(conn *ggtcp.Conn) {
}))

app := dove.New(
    dove.Name("dove"),
    dove.Version("v1.0.0"),
    dove.Server(tcp),
)

if err := app.Run(); err != nil {
    log.Fatalln(err)
}
```
