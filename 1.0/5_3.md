# TCP 服务器

server/gtcp 中基于 G 包 TCP 组件实现了 Server，用以注册 tcp 到 dove.Server() 中。

## 服务注册

```go
import (
    ggtcp "github.com/camry/g/gnet/gtcp"
)

tcp := gtcp.NewServer(gtcp.Handler(func(conn *ggtcp.Conn) {
}))

app := dove.New(
    dove.Name("dove"),
    dove.Version("v1.0.0"),
    dove.Server(tcp),
)

if err := app.Run(); err != nil {
    log.Fatalln(err)
}
```
