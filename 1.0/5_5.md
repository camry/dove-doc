# CRON 服务器

CRON 服务器。

## 注册服务

```go
gc := gcron.NewServer(
    gcron.Options(cron.WithSeconds()),
    gcron.Logger(glog.GetLogger()),
)

// gc.AddJob("* * * * * *", ...)

app := dove.New(
    dove.Name("dove"),
    dove.Version("v1.0.0"),
    dove.Server(gc),
)

if err := app.Run(); err != nil {
    log.Fatalln(err)
}
```
