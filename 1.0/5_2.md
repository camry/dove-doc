# GRPC 服务器

server/grpc 中基于 Google gRPC 框架实现了 Server，用以注册 grpc 到 dove.Server() 中。

## 注册服务

```go
gs := grpc.NewServer(
    grpc.Address(":8002"),
)

app := dove.New(
    dove.Name("dove"),
    dove.Version("v1.0.0"),
    dove.Server(gs),
)

if err := app.Run(); err != nil {
    log.Fatalln(err)
}
```
