# UDP 服务器

UDP 服务器。

## 注册服务

```go
import (
    ggudp "github.com/camry/g/gnet/gudp"
)

udp := gudp.NewServer(gudp.Handler(func(conn *ggudp.Conn) {
}))

app := dove.New(
    dove.Name("dove"),
    dove.Version("v1.0.0"),
    dove.Server(udp),
)

if err := app.Run(); err != nil {
    log.Fatalln(err)
}
```
