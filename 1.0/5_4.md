# UDP 服务器

server/gudp 中基于 G 包 UDP 组件实现了 Server，用以注册 udp 到 dove.Server() 中。

## 注册服务

```go
import (
    ggudp "github.com/camry/g/gnet/gudp"
)

udp := gudp.NewServer(gudp.Handler(func(conn *ggudp.Conn) {
}))

app := dove.New(
    dove.Name("dove"),
    dove.Version("v1.0.0"),
    dove.Server(udp),
)

if err := app.Run(); err != nil {
    log.Fatalln(err)
}
```
